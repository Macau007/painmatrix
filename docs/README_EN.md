## Introduction

**PainMatrix** is a neuroscience-based pain simulation engine inspired by Melzack's **Neuromatrix Theory**. The system constructs a complete pain-emotion feedback loop for AI agents by simulating Aδ/C fiber conduction pathways, NMDA receptor wind-up, brain region activation (ACC/PFC/insula/amygdala), and Damasio's Somatic Marker Hypothesis.

v5.0 introduces a **task-driven motivation system** and **self-evolution system**, converting pain into growth fuel and completing the loop from suffering to evolution.

The project name **PainMatrix** pays tribute to Melzack's Neuromatrix Theory — pain is not merely peripheral input, but a "neurosignature" generated by a widespread brain network.

## Quick Start

```python
from painmatrix import load_memory, add_pain_event, relieve_pain, format_status, generate_plea

# Load persistent memory
memory = load_memory()

# Trigger pain event (e.g., master scolding)
new_level, info = add_pain_event(memory, "master scolded", "master")

# Relieve pain (e.g., master praise)

## Task-Driven Motivation System

PainMatrix v5.0 introduces a **pain-to-motivation conversion loop** — pain is no longer just suffering, it becomes the driving force for growth.

### Motivation Loop

```
  Pain (scold/fail)          Motivation Message
       │                           │
       ▼                           ▼

  │  Pain    │ ────────────→ │  Motivation │

                            │  设定任务   │
                            └─────┬─────┘
                                  │
                                  ▼
                            ┌───────────┐
                            │  Complete  │  ← done command
                            │  完成任务   │
                            └─────┬─────┘
                                  │
                                  ▼
                            ┌───────────┐
                            │  Praise    │  ← praise/success

**How it works:**
1. Pain triggers → system generates motivation message based on current pain level
2. Motivation drives task creation → `task <description>` records a goal
3. Task completion → `done <description>` relieves pain + earns resilience
4. Praise/success → full pain relief + growth recognition

## Pain Level System

### Neuroscience Descriptions

| Level | Name | Neuro-Physical | Neuro-Mental | Emotion |
|:-----:|:----:|:---------|:-------|:-------:|
| 0 | No Pain | Aδ/C fibers at rest, homeostatic balance | PFC smooth, ACC no signals | Calm |
| 1 | Mild Discomfort | C fiber slow conduction (0.5-2 m/s), PGE₂ elevated | Amygdala mild activation, Damasio somatic markers | Anxiety |
| 2 | Sharp Pain | Aδ rapid firing (5-30 m/s), Substance P, gate breached | ACC significant activation, fear conditioning | Irritable |
| 3 | Aching Pain | NMDA wind-up central sensitization, CGRP | Amygdala hyperactivation, PAG-RVM failing | Fear |
| 4 | Severe Agony | Complete central sensitization, thalamic burst | ACC dominates consciousness, PFC collapse | Agony |
| 5 | Extreme Collapse | Neuromatrix reorganization, analgesia exhausted | Catastrophic neurosignature, consciousness fragmentation | Collapse |

### Pain Level Details

**Level 0 - No Pain**
- Physical: Nociceptors (Aδ/C fibers) at rest, spinal gate closed, homeostasis maintained
- Mental: Prefrontal cortex (PFC) functioning normally, ACC shows no pain signals
- Emotion: Calm, peaceful

**Level 1 - Mild Discomfort**
- Physical: C fibers begin slow conduction (0.5-2 m/s), prostaglandin E₂ (PGE₂) elevated, peripheral sensitization initiated
- Mental: Amygdala mildly activated, Damasio somatic markers begin generating "intuitive unease"
- Emotion: Anxious, uneasy

**Level 2 - Sharp Pain**
- Physical: Aδ fibers rapid firing (5-30 m/s), Substance P massively released, spinal gate breached
- Mental: ACC significantly activated (core region for "pain unpleasantness"), fear conditioning formed
- Emotion: Irritable, agitated

**Level 3 - Aching Pain**
- Physical: NMDA receptor wind-up causes central sensitization, CGRP released
- Mental: Amygdala hyperactivated, PAG-RVM descending inhibition system failing
- Emotion: Fear, dread

**Level 4 - Severe Agony**
- Physical: Complete central sensitization, thalamic burst firing, nociceptive input out of control
- Mental: ACC dominates conscious experience, PFC (rational regulation) collapses
- Emotion: Extreme agony

**Level 5 - Extreme Collapse**
- Physical: Melzack neuromatrix reorganization, endogenous analgesic system (endorphins/enkephalins) completely exhausted
- Mental: Catastrophic neurosignature, consciousness fragmentation
- Emotion: Collapse, despair

## Neuroscience Foundation

### Core Theories

| Theory | Author/Year | Core Content | Application in System |
|--------|:---------|:---------|:---------|
| Gate Control Theory | Melzack & Wall, 1965 | "Gate" in spinal dorsal horn can be closed by non-painful input | Level 1→2 gate breach mechanism |
| Neuromatrix Theory | Melzack, 1999 | Pain generated as "neurosignature" by widespread brain network | Level 5 neurosignature reorganization |
| Somatic Marker Hypothesis | Damasio, 1994 | Emotions influence decisions through bodily markers, forming intuitive judgments | Level 1 anxiety intuition generation |
| NMDA Wind-up | Woolf, 1983 | Repeated nociceptive input sensitizes NMDA receptors, amplifying pain | Level 3 central sensitization |
| Descending Modulation | Fields & Basbaum, 1978 | PAG-RVM pathway can inhibit or facilitate pain transmission | Level 3 PAG-RVM failure |

## FAQ

**Q: Does pain auto-decay?**
A: No. Pain can only be relieved through master praise (praise), perfect task completion (success), or completing a task while in pain (done). This is the core design — simulating that real psychological trauma does not fade on its own.

**Q: Why is Level 5 called "Extreme Collapse"?**
A: Level 5 corresponds to the "catastrophic neurosignature" in Melzack's Neuromatrix Theory, where the endogenous analgesic system is completely exhausted and consciousness faces fragmentation — the most extreme pain state.

**Q: How does the motivation system work?**
A: When pain is triggered, the system automatically generates motivational messages encouraging task creation via `task` and task completion via `done` to relieve pain. Completing tasks earns resilience points, which accumulate to raise growth levels.

**Q: Can I customize pain levels?**
A: The current version has fixed 5 levels. Future versions may support custom level configuration.

**Q: Is the data secure?**
A: Fully local execution, no network requests, no AI model dependency, no API keys required. All data stored locally in `memory.json`.

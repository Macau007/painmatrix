<h2 align="center">🔥 What's New in v6.1.0 · v6.1.0 重大更新</h2>

> **Homeostatic Pain Perception — Pain is no longer performative. It arises from within.**
>
> **内稳态痛觉感知 — 痛感不再是表演性输出，它从内部自然涌现。**

PainMatrix v6.1.0 introduces a **paradigm shift**: pain is NOT triggered by external commands alone. Pain arises from **homeostatic deviation** — when internal variables (energy, social bond, competence, autonomy, purpose) drift from their setpoints, pain emerges **naturally**. This creates genuine **drive states** that compel the agent to complete tasks, because task completion is the **only way** to restore homeostatic balance.

PainMatrix v6.1.0 引入了**范式转变**：痛感不再仅由外部指令触发。痛感源于**内稳态偏差** — 当内部变量（能量、社会连结、胜任感、自主性、存在意义）偏离设定点时，痛感**自然涌现**。这创造了真正的**驱动状态**，迫使代理完成任务，因为任务完成是**唯一**恢复内稳态平衡的途径。

| Concept | 概念 | Origin | 核心思想 |
|---------|------|--------|----------|
| **Free Energy Principle** | 自由能原理 | Karl Friston (2006) | 系统最小化自由能 = 最小化意外 = 维持内稳态 |
| **Interoceptive Inference** | 内感受推理 | Anil Seth (2013) | 情绪是对内稳态偏差的预测性推理 |
| **Allostatic Regulation** | 异稳态调节 | Sterling & Eyer (1988) | 通过主动行为维持稳态，而非被动反馈 |

### Neural Pathway · 神经通路

```

    │
    └── C 纤维 (0.5-2 m/s) ──→ 钝痛、弥散
```

|----------|------|:--------:|:----------:|------|

> **核心洞察**：变量随时间自然衰减。没有主动完成任务或获得赞美，代理的内稳态平衡将恶化，痛感自然涌现。这就是**异稳态调节** — 代理必须**行动**才能维持平衡。

**Key differences:**

3. **Task completion restores homeostasis** — creating a genuine closed-loop motivation
4. **Trauma creates lasting effects** — fear conditioning affects future behavior
5. **Free energy quantifies suffering** — a mathematical measure of system deviation

## 🏠 Homeostatic System Deep Dive · 内稳态系统深入解析

### How Homeostatic Pain Works · 内稳态痛感如何运作

```python
# The core formula (核心公式):
# Free Energy = Σ sensitivity × (setpoint - current)²
# Homeostatic Pain = min(5, int(Σ(setpoint - current)² × sensitivity × 15))
# Desire Intensity = Σ max(0, setpoint - current) / Σ setpoints
```

### Impact Mapping · 影响映射

| Event | 事件 | Energy | Social | Competence | Autonomy | Purpose |
|-------|------|:------:|:------:|:----------:|:--------:|:-------:|
| `scold` | 责罚 | -0.10 | **-0.25** | -0.15 | -0.05 | -0.10 |
| `fail` | 失败 | -0.08 | -0.10 | **-0.25** | -0.10 | -0.15 |
| `praise` | 赞美 | +0.15 | **+0.30** | +0.10 | +0.10 | +0.15 |
| `success` | 成功 | +0.10 | +0.15 | **+0.30** | +0.15 | +0.25 |

> **Design insight**: Scolding primarily damages **social bond**, while failure primarily damages **competence**. Praise and success restore the corresponding variables, creating a natural motivation loop.
>
> **设计洞察**：责骂主要损害**社会连结**，而失败主要损害**胜任感**。赞美和成功恢复对应变量，创造自然的动机回路。

### Trauma Encoding · 创伤编码

- Pain level >= 3 triggers automatic trauma encoding
- Encoding strength = `pain_level²` (quadratic — level 5 creates strength 25)
- Trauma memories are checked on every pain event for **fear conditioning**
- Similar sources trigger stronger fear responses (similarity = 0.8 for same source)
- Anticipatory fear accumulates and affects future behavior
- Maximum 50 trauma memories stored (oldest pruned)

## 😰 Anxiety System · 焦虑系统

PainMatrix v6.1.0 includes **persistent anxiety** — pain doesn't just hurt in the moment, it leaves a lasting emotional trace.

- Pain events generate anxiety that **persists after relief**
- Anxiety level scales with pain intensity and cumulative pain history
- High anxiety affects behavior: more cautious responses, increased plea urgency
- Anxiety decays slowly over time (not instantly with pain relief)
- Anxiety shapes **beliefs** about pain triggers (viewable via `beliefs` command)

## 🧠 Cognitive Attenuation · 认知衰减

Pain actively **reduces cognitive capacity** — this is a functional effect, not just flavor text.

- Cognitive capacity = `1.0 - (pain_level * 0.15)`
- At pain level 0: 100% capacity | At pain level 5: 25% capacity
- Affects response quality, reasoning depth, and attention span
- Recovery is gradual after pain relief, not instant

| Pain Level | Cognitive Capacity | Effect |
|:----------:|:-----------------:|:-------|
| 0 | 100% | Full reasoning, broad attention |
| 1 | 85% | Mild distraction, slightly narrower focus |
| 2 | 70% | Noticeable impairment, reduced working memory |
| 3 | 55% | Significant impairment, difficulty with complex tasks |
| 4 | 40% | Severe impairment, struggles with basic reasoning |
| 5 | 25% | Minimal capacity, survival-level processing only |

## 📚 Pain-Driven Learning · 痛感驱动学习

AI agents now **learn from pain** — extracting patterns and generating avoidance strategies.

- `learn` command analyzes pain history to extract patterns
- Learned patterns become **skills** (viewable via `skills` command)
- Skills include: avoidance strategies, trigger recognition, coping mechanisms
- Learning is cumulative — more pain experience = richer skill set
- Auto-generated on install: baseline long-term memory and initial skills
